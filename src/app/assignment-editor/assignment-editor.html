Max Element in Array

Find the maximum element from the given array of integers.

Input Format

First line contains N.
Second line contains N integers.

Output Format

Print the maximum element.

Constraints 

1 ≤ N ≤ 10³

-10⁹ ≤ arr[i] ≤ 10⁹

Sample Input 

5
-2 -19 8 15 4

Sample Output 

15

Explanation

Among all numbers, 15 is the largest element.
